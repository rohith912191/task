# Finance Dashboard UI

A polished React dashboard built for clarity, real-world finance monitoring, and role-aware behavior.

## Overview
This repository implements a frontend-only finance dashboard with a thoughtful layout and professional interaction model:

- Summary cards for balance, income, and expense totals
- Time-based trend visualization for recent cash flow
- Category spending breakdown with ratio insight
- Transactions table with search, filter, and sort controls
- Role switcher (`Viewer` / `Admin`) to simulate read-only and data-management personas
- Insights panel with highest spending category and month-over-month comparison
- Responsive layout, dark mode, and local persistence for user preferences

## Features

- `Viewer` role explores data without editing
- `Admin` role can add new transactions dynamically
- Application state is managed with React hooks and memoized derived data
- Data-driven UI avoids blank or ambiguous displays
- Clean, maintainable component structure suitable for an experienced frontend developer

## Setup

1. Open the project folder in your terminal.
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the local Vite URL shown in the terminal.

## Notes

- This dashboard is intentionally static and frontend-driven.
- Dark mode preference and selected role are persisted in local storage.
- Designed for readability, simplicity, and a clean modern UI.
